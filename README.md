# Environment
This project requires the following environment variables be set:
* For database image:
    * ENV['POSTGRES_USER']
    * ENV['POSTGRES_PASSWORD']
* To set author
    * ENV['PRIMARY_USER']
    * ENV['PRIMARY_USER_PW']
